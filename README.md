## 📌 Overview

This project is a command-line AI Chatbot developed in Python. It can respond to common user messages and demonstrates the fundamentals of chatbot development using Python.

---

## ✨ Features

- 💬 Interactive conversation
- 👋 Greets users
- 🤖 Responds to common questions
- 🚪 Exit using the `bye` command
- 🐍 Beginner-friendly Python project

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Chatbot.git
```

2. Open the project folder

```bash
cd AI-Chatbot
```

3. Run the chatbot

```bash
python chatbot.py
```

## 💻 Example

```text
You: hello
```
